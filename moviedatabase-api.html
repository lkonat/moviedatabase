Movie Database

This program is a helpful tool to help actors and directors to collaborate for projects and future projects. Each actor should make a profile, with the options of measurements and adding 5 profile pictures. Directors can search profiles, make a list, and then save, email, or text the list to anyone they choose. Each user will log in with their email and have a password, with the option to reset their password if forgotten.
